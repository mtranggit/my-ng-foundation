Demo card layout in flexbox mode

Tall Card

I'm the tallest card in the row. By default, all the other cards in the same row will be only as tall as the content inside of them. However, making their container, display: flex will make sure the column in the row are the same height. All this will also allows the cards inside those columns to grow to the height of the column.

Shorter card

I'm the shorter card. Adding the .flex-container class to the card's parent (in this case the column) will make it display flex and therefore share the same height as other columns in its row.

Shortest Card

I'm the shortest card in the row. I don't have much content here but I'm still equal height.

Cards

A card is just an element with a .card class applied. You can put any kind of content inside. Make sure you wrap your content in a .card-section element in order to achieve the traditional card look.

A card container has no padding, allowing you to place full-bleed images inside. Use the .card-divider and .card-section classes to sub-divide a card.
